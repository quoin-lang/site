inherits Object

A 64-bit IEEE-754 floating-point number -- the type of literals like 3.14. Arithmetic with either an Integer or a Double operand happens in floating point and yields a Double. Whole values print without a decimal point (4.0.s is '4').

Infinity and NaN follow IEEE rules: 1.0 / 0.0 is Double.inf, 0.0 / 0.0 is Double.nan, and NaN compares unequal to everything, itself included.

0.1 + 0.2     "* -> 0.30000000000000004
1.0 / 0.0     "* -> inf
